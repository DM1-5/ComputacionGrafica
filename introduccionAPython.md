# Introduccion a python

## Tipos de variables

- *Enteros*
- *Float*
- *Strings*
- *boolean*

La palabra reservada input() se utiliza para tomar informacion del usuario.
Al agregar el input la variable se debe convertir al tipo de dato porque la informacion dada se toma como string

## Conjuntos de datos

- *listas* = [ ]
- *tuplas* = ( )
- *llaves* = { }

## Operadores matematicos

Las listas son modificables, las tuplas no son modificables.

- *Suma* +
- *Resta* -
- *Multiplicacion* *
- *Potenciacion* **
- *Division* /
- *Division entera* //

### Libreria de matematica (math)

- *Raiz cuadrada* sqrt()
- *math.sin() math.cos()* (Los grados van a estar en radianes)
- *math.radians() math.degrees()* (para convertir entre grados y radianes)
